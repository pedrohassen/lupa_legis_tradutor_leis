# lupa_legis_tradutor_leis
Este projeto consiste em uma Prova de Conceito (PoC) acadêmica voltada a testar a viabilidade técnica e prática do sistema Lupa Legis. A finalidade central é atuar como uma ferramenta de simplificação e resumo de textos legislativos, tornando-os compreensíveis para o cidadão comum.
